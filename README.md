# 🦑 Squid Game: Red Light Green Light — 3D Browser Arena

A full cinematic recreation of Squid Game's iconic "Red Light Green Light" challenge — built entirely in the browser using React Three Fiber and meme-powered Tamil visuals. You move, you risk elimination. You win, you bask in thalluvathi glory.

---

## 🎮 Live Experience

Players spawn, run toward the finish line, and freeze when the doll turns. Move during red light? You're eliminated — with visual texture swaps, audio scares, and real-time feedback.

---

## 🚀 Tech Stack

- ⚛️ React + React Three Fiber (`@react-three/fiber`)
- 🎨 Drei components (`<Text>`, `<Environment>`, `<Suspense>`)
- 💾 GLB models (custom doll & background)
- 🔊 Audio integration triggered by game phases
- 🖼️ Image texture swaps for player avatars
- 🔧 Dynamic elimination, movement, and winner logic

---

## 💡 Features

- 🧒 Rotating doll — flips between red/green light
- 📦 Image-based player boxes — safe and eliminated textures
- 🔴 Eliminated players freeze and turn red
- 🏁 Start/Finish lines with Tamil labeling
- 🎧 Sound effects triggered by game flow
- 🧠 Status overlays above players
- 🕹️ Scalable architecture for multiplayer expansion
